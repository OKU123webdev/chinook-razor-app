# Chinook Razor Application

This project is a **C# ASP.NET Core Razor Pages** web application demonstrating database-driven functionality using the Chinook sample dataset.

## Key Features
- Create, read, update and delete database records
- Sort data in ascending and descending order
- Filter data by category
- Search database records
- Input validation

## Technology Stack
- Language: C#
- Framework: ASP.NET Core (Razor Pages)
- Database: SQLite (Chinook sample database)
